对windows系统的ipv6功能的开启

使用 netsh 命令关闭 IPv6 隧道适配器：

netsh interface teredo set state disable
netsh interface 6to4 set state disabled
netsh interface isatap set state disabled

当我们需要使用 IPv6 网络时，只需要使用如下命令恢复 IPv6 隧道设置即可：

netsh interface teredo set state default
netsh interface 6to4 set state default
netsh interface isatap set state default

具体的设置还有很多，一般来说，windows7或者windows10通过上述命令在powershell上逐条执行，就可以使用ipv6了

对于windowsXP还需要安装一下ipv6的网络协议。当然这种办法还有一个问题，有时候在每一次电脑重新启动的时候，会自动把已经开启的这些隧道给自动关闭，需要在组策略上进行设置。当然这个没关系的。就是把这三条命令重新复制粘贴一下就可以。不废什么事情的。

PS：有时候会在cmd的命令行下，会提示没有权限 ，这个时候，需要以管理员的权限执行。只需要找到C:\windows\system32\cmd.exe，然后再cmd.exe右键以管理员的形式运行。
